# 🍽️ Recipe App

A modern recipe discovery web application built with React and Tailwind CSS. Users can browse, search, and explore a variety of recipes through a clean, responsive, and user-friendly interface.

## 🚀 Features

- Search recipes instantly
- Browse recipes by category
- View detailed recipe information
- Responsive design for all devices
- Modern and intuitive UI
- Fast and optimized performance

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- REST API

## 📸 Preview

A clean and modern recipe platform for discovering delicious meals and cooking inspiration.

## ⚙️ Installation

```bash
git clone <repository-url>
```

```bash
cd recipe-app
```

```bash
npm install
```

```bash
npm run dev
```
